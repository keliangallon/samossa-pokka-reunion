# 🥟 Classement Samossas & Pokkas — Île de La Réunion

Application web pour classer et noter les meilleurs samossas et pokkas de La Réunion.

**→ [Voir l'application en ligne](https://TON-PSEUDO.github.io/samossa-pokka-reunion/)**

## Fonctionnalités

- 🥟 Classement des samossas par note moyenne
- 🥙 Classement des pokkas par note moyenne  
- 🗺️ Carte de La Réunion avec les vendeurs de samossas
- 👥 Mode multijoueur — chaque joueur vote séparément
- ➕ Ajout de nouveaux samossas / pokkas
- 🏆 Podium automatique mis à jour en temps réel

## Lancer en local

```bash
npm install
npm run dev
```

## Déployer

Le déploiement sur GitHub Pages est automatique à chaque `push` sur `main` via GitHub Actions.

## Stack

- React + Vite
- GitHub Pages (hébergement gratuit)
- GitHub Actions (CI/CD automatique)
